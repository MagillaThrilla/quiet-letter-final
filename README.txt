LOCAL TEST
Double-click index.html.

For a more accurate browser-server test:
1. Open Terminal.
2. Drag the project folder into Terminal after typing: cd 
3. Press Return.
4. Run: python3 -m http.server 8000
5. Open: http://localhost:8000
6. Stop with Control+C.

GITHUB PAGES
1. Place index.html, style.css, and the assets folder in the repository root.
2. Commit and push using GitHub Desktop.
3. On github.com, open the repository.
4. Open Settings > Pages.
5. Under Build and deployment, choose Deploy from a branch.
6. Select main and / (root), then Save.
